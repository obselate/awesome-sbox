# Want to contribute?

If you would like to add a project to the awesome-sbox repository, fork this project and create a pull request with your changes.

If your PR does not meet the guidelines, it will be rejected with comments.

## Guidelines

Your addition to the **awesome-sbox** repository must be:
1. In the correct category / subcategory
2. Use markdown inline link syntax eg. \[Example]\(https://google.com)
3. Use markdown table syntax (eg. | Link goes here | Description goes here |) and provide an accurate description of the linked content
4. An open source project that is actively updated or documentation that is current

If a category for your listing does not exist, include it in your PR. Please try to stick to the provided categories if possible.
