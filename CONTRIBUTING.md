# Want to contribute?

If you would like to add a project to the awesome-sbox repository, fork this project and create a PR with the changes.

## Guidelines

Your addition to the **awesome-sbox** repository must be:
1. In the correct category
2. Use markdown inline link syntax eg. \[Example]\(https://google.com)
3. An open source project on github that is actively updated

If a category for your listing does not exist, include it in your PR. Try to stick to the provided categories if possible.
